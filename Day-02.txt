1) 5 differences between BrowserJS and NodeJS

a)Javascript is basically used on the client-side while NodeJS is used on the server-side.
b)Javascript is capable enough to add HTML and play with DOM while NodeJS is not capable of adding HTML tags.
c)Javascript runs any engine like Spider monkey (Firefox), JavaScript Core (Safari), V8 (Google Chrome) accordingly to the browser while Node.js runs in a V8 engine which is mainly used by google chrome.
d)Node.js is headless i.e without any GUI while Bowsers are not headless.
e)In Node.js everything is a module i.e it is mandatory to keep everything inside a module while moduling is not mandatory for browser javascript.
